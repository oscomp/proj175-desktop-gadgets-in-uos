基于uos操作系统，实现一套桌面小部件，并允许用户自定义扩充。

## 预期目标

### 题目中预期目标需全部达成，附加题属于额外得分

1、使用qt+qml进行开发
2、默认提供天气，时钟，便签三个部件
3、部件相对大小为以下可能中的几种或全部1x1,1x2,1x4,2x2,2x4,3x4,4x4
4、支持拖拽预览区域的部件到常用区域，部件拖拽流畅且自然，无卡顿
5、支持部分小部件调整大小，调整后也应该为目标3中的尺寸中的一个

### 附加目标

1、提供小部件开发手册，允许第三方按照规范开发自己的小部件
